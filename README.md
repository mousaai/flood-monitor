# FloodSat AI — منصة الرصد الذكي للفيضانات | أبوظبي

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/deploy?template=https://github.com/mousaai/flood-monitor)

> منصة ذكية لرصد الفيضانات في إمارة أبوظبي بالوقت الفعلي، مدعومة بالذكاء الاصطناعي وبيانات الأقمار الاصطناعية.

## 🌐 الموقع الحي

**[https://flood-monitor-app-production.up.railway.app](https://flood-monitor-app-production.up.railway.app)**

## 🚀 نشر المشروع

انقر على الزر أعلاه لنشر نسختك الخاصة على Railway بنقرة واحدة.

## 🛠️ التقنيات المستخدمة

- **Frontend:** React + TypeScript + TailwindCSS + Vite
- **Backend:** Express + tRPC
- **البيانات:** Open-Meteo ERA5 API (بيانات حية)
- **الاستضافة:** Railway

## 📋 الميزات الرئيسية

- لوحة قيادة رئيسية مع مؤشرات KPI حية
- خرائط تفاعلية موحدة
- نظام إنذار مبكر مع 86+ تنبيه
- استكشاف 90 منطقة في الإمارة
- محاكاة سيناريوهات الفيضانات
- دعم متخذي القرار
- تحليل صور ميدانية بالذكاء الاصطناعي
- أرشيف أحداث 2011–2024
- تقارير قابلة للتصدير PDF
- دعم اللغتين العربية والإنجليزية (RTL/LTR)
